nestemice — nový web

Soukromý projekt FC. Statický web, připravený na hosting přes Netlify/Vercel/GitHub Pages.

Stránky

  • index.html — úvod + hero feed z deníku
  • denik.html — deník psaný AI, filtry podle tagu
  • kronika.html — galerie 189 listů (placeholdery, očekává grafika/kronika1/N.jpg)
  • zub-casu.html — 3 akty × 5 trojic (1901 · 2001 · 2026)
  • o-projektu.html — lidé, zdroje, kontakt
  • archiv-2001.html — replika původního webu jako muzejní exponát

Deník pipeline

Seznam článků je v denik/index.json. Plný text článku v denik/clanky/YYYY-MM-DD-slug.md. Detailní instrukce pro AI agenta v denik/README.md.

Lokální náhled

cd projekt
python3 -m http.server 8000
# otevři http://localhost:8000

Přímé otevření přes file:// nezobrazí deník — prohlížeč blokuje fetch JSONu. Musí běžet přes HTTP server (výše nebo Netlify/GitHub Pages).

Co doplnit reálnými daty

  • grafika/kronika1/0.jpg … 198.jpg — 189 listů
  • grafika/zub-casu/akt1/01-{1901,2001,2026}.jpg atd. — 15 trojic fotografií
  • historie/*.html — 4 kronikářské stránky (Ryjice, Blansko, doprava, fotbal) — odkazy připravené, stránky zatím 404
  • Skutečný kontaktní e-mail v o-projektu.html a shell.js

Deploy

Stačí pushnout do GitHub repa a na Netlify / Vercel / GitHub Pages připojit. Žádný build step. Později doménu směrovat A/CNAME rekordem k hostingu.
